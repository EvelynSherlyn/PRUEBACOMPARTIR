## 🏷️ || Descripción
Este código sacado de un kata de codewars. Consiste en tomar una String por parámetro y devolver una String en Camel Case. Ideal para practicar Strings.



## 🌟 || Ejemplos
Entrada: `hola-Don_pepito`  
Salida: `HolaDonPepito`
